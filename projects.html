🎤 Qwen3-TTS Enterprise Gateway (AICESS)

Period: 2025.04 ~ Present

Role: 혼자 풀스택 설계 및 구축 (Architecture, Backend, Frontend, DevOps)

시스템 아키텍처

FastAPI 기반 WebSocket/REST API 게이트웨이, vLLM-Omni(Qwen3-TTS-1.7B) 추론 엔진, Redis 분산락 & 캐싱 레이어, React 관리자 대시보드. 문장 단위 논리 청킹, 지속적 WebSocket 스트리밍, Pre-Normalization으로 초저지연 구현.

성과 (KPI)

  • ✓ TTFB: 2292ms → 334ms (85% 단축, 6.85배 개선)
  • ✓ 동시 처리 채널: 5개 → 12개 (240% 확대)
  • ✓ 초저지연 스트리밍: 500ms 이내 첫 오디오 출력
  • ✓ 프로덕션 환경: 월 10,000+ 요청 처리, 99.5% 가용성

핵심 엔지니어링 (5가지 문제 해결)

  1. 스트리밍 오디오 팝 노이즈 제거 – Residual Buffer 설계로 PCM 얼라인먼트 해결, Micro-fade 적용
  2. 스토리지 누수 방지 – Deep GC 추적기 구현으로 MD5 해시 캐시 파일 완벽 정리
  3. 웹소켓 세션 탈취 방어 – password_changed_at 검증으로 제로-트러스트 모델 구현
  4. 정규식 토크나이저 고도화 – 스마트 ITN 엔진 개발로 URL/이메일 자연스럽게 처리
  5. CPU 병목 제거 – ThreadPoolExecutor 오버헤드 제거, O(N²) → O(1) 성능 개선
